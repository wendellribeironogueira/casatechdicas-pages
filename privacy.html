Política de Privacidade

Esta política descreve como tratamos suas informações ao usar o sistema CasaTechDicas.

1. Coleta de Dados

O CasaTechDicas não coleta dados pessoais diretamente dos usuários. O sistema opera exclusivamente com dados públicos ou fornecidos manualmente por você em planilhas ou arquivos locais.

2. Informações Processadas

Os únicos dados manipulados são produtos afiliados, links de redirecionamento, títulos e conteúdos gerados por IA. Esses dados permanecem sob seu controle e não são enviados a servidores externos, exceto APIs públicas como Shopee ou TikTok.

3. Segurança

Todos os scripts rodam localmente ou em ambientes autorizados pelo usuário. Nenhuma informação é armazenada ou compartilhada com terceiros sem seu consentimento.

4. Terceiros

Este projeto pode integrar serviços de terceiros (APIs de afiliados, redes sociais, ferramentas de análise), cujas políticas de privacidade são independentes. Recomendamos revisar os termos de cada um separadamente.

5. Contato

Para dúvidas sobre esta política, entre em contato com o administrador responsável pelo projeto CasaTechDicas.
